patchbin

by jbritain

a utility for generating binary patches that can be used to reconstruct a modified version of a file without directly distributing that modification

if you are looking to generate a patch file to distribute, upload the original file and your modified version, and click "generate patch". you can then safely distribute the downloaded '.bin' file without violating any licensing agreements.

if you have been given a patch file, upload the original file and the patch file, and click "apply patch".

if you find this utility useful, consider donating to help me keep the lights on.




Patch generation is hidden by default to make this tool more accessable to users.
